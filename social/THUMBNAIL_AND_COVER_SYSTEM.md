# Thumbnail and Cover System
> **Internal draft only.** No publication, outreach, live connection, spend, or asset use is authorized. Verify claims and record permissions before use.

## Recognition recipe
Real floor texture occupies at least two-thirds of frame; warm wood/neutral grade; charcoal or cream title; thin accent rule; small upper-left franchise tab; optional small approved wordmark. This is a proposed visual system pending brand assets and accessibility review.

| Franchise | Cover label | Image rule |
|---|---|---|
| Saved? | CAN THIS BE SAVED? | Damage close-up, no reveal deception |
| Floor Crimes | FLOOR CRIME: [ISSUE] | Anonymous defect only |
| Explain | FLOOR GUYS EXPLAIN | Tool/material/detail tied to question |
| Rescue | FLOOR RESCUE | Honest aligned split or strongest real condition |
| History | NOLA FLOOR HISTORY | Verified floor/building detail only |
| Field Notes | CONTRACTOR FIELD NOTE | Plan/detail with private data removed |
| Cost | WHAT AFFECTS COST? | Scope variable, never dollar clickbait |
| ASMR | FLOOR ASMR | Texture/process macro |
| Friday | FINISH FRIDAY | Final detail; “before” badge if carousel |

## Rules
Three-to-six words, one promise, mobile legibility, subject not covered, no fake flames/arrows/shock faces. Export source plus no-text version; log asset ID. A/B testing changes one element only and never uses misleading crops.
