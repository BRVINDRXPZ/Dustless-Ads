# Prohibited and High-Risk Content
> **Planning only.** No publication, outreach, live system connection, campaign creation, spend, or use of uncleared proof is authorized.

## Never post
Fabricated/AI project proof; fake reviews/results/credentials/prices/statistics; customer/address/private information; competitor or customer humiliation; unsafe demonstrations; dust-free claims; unsupported “best/#1/most trusted”; fake scarcity; review gating/manipulation; copyright infringement; unlicensed music/footage; undisclosed endorsements; altered before/after color/condition; random trends/filler; discriminatory or harassing content; sensitive security/access details.

## Mandatory hold/review
| Content | Required clearance |
|---|---|
| Historic/species/material claim | Source citation + qualified reviewer + project verification |
| Product, cure, odor, durability, safety | Current manufacturer documents/SDS as applicable + technical/legal review |
| Named project, exterior, neighborhood, drone | Property/customer/location/airspace and platform rights; privacy review |
| Testimonial/reaction | Exact quote/outcome, person identity, organic/paid usage and withdrawal permission |
| Commercial/contractor capability | Relevant actual case/process/compliance proof |
| Cost/offer/schedule/service area | Owner-approved current terms, capacity and disclaimer |
| Reaction content | Transformative commentary, minimal necessary excerpt, rights/fair-use review; licensed source preferred |
| Children, workers, bystanders | Explicit appropriate releases; otherwise exclude |
| Complaint/incident | Owner/legal/insurance pathway; never content-farm conflict |

## Stop-work triggers
Unclear consent, active dispute, injury/unsafe condition, private information, claim conflict, inaccurate color, copyrighted source uncertainty, requested revocation, or pressure to stage a result. Quarantine the asset and escalate.
