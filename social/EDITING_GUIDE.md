# Editing Guide
> **Internal draft only.** No publication, outreach, live connection, spend, or asset use is authorized. Verify claims and record permissions before use.

| Element | Standard |
|---|---|
| Masters | Archive clean 4K 16:9 and/or 9:16 source-quality master; retain captions/music/logo as separate layers |
| Ratios | 9:16 shorts/Stories; 4:5 feed; 1:1 only where needed; 16:9 YouTube long-form |
| Length | ASMR 8–20s; explainers 20–45s; transformations 45–90s; long-form 5–12m when justified |
| Hook | Visual condition/result in first 1–2s; context immediately; no deceptive withheld reveal |
| Pacing | Cut on meaning/action; let craft shots breathe 1–3s; remove dead air, not honest context |
| Captions | Accurate burned/open captions; sentence case; high contrast; correct local/product terms manually |
| Text safe area | Keep critical text away from top/bottom UI and edges; test on device; 3–6 words per cover line |
| Fonts/colors | Use approved brand font/colors when supplied; until then use a consistent accessible sans-serif and neutral charcoal/cream/wood palette as a placeholder |
| Logo | Small end/cover mark; never obscure work; no constant oversized bug |
| Music/SFX | Licensed for destination/use or platform-cleared; store license; natural sound preferred; no unlicensed trending audio in ad masters |
| Color | Correct neutrally; preserve grain, stain, sheen; no selective enhancement; match before/after honestly |
| Transitions | Straight cuts/dissolves; split-screen only aligned; no effect that conceals discontinuity |
| VO/audio | Direct local expert delivery; clean levels; duck music; preserve safe equipment sound; disclose reconstructed audio if ever used |
| Thumbnail/outro | Series cover system; 0.5–1.5s end card with one CTA; no fake reaction face |
| Export | H.264 high-quality, AAC; platform specs validated at export time; review captions/crop/color/audio after upload preview |
| QA | Frame-by-frame privacy; claims; permission; spelling; continuity; music rights; CTA/link; no watermark |
