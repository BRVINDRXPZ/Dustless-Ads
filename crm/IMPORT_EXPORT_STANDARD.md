# Import Export Standard

> **Planning specification — inactive.** No live accounts, credentials, communications, or real customer data.

## Purpose

UTF-8 CSV, canonical headers, ISO dates/times, decimal USD, immutable IDs, manifest with source/run/schema version/hash/row count. Dry-run validation and quarantine before import; reversible batch ID; exports minimize PII and include filter/time/freshness. Never include secrets.

## Control

Owner approval, audit logging, least privilege, and documented exception handling are mandatory.
