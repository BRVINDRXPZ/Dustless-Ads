# Local Link Building

## Principle
Earn editorial references by being useful and having a real relationship, project, resource, membership or newsworthy contribution. Referral and brand value matter even when a link is unavailable. No reciprocal-link quota, anchors, payment for ranking, disguised sponsorship, mass outreach or fabricated affiliation.

## Offer menu
Permissioned technical guide; expert-reviewed preservation/floor-care session; project case study; co-created planning checklist; contractor coordination resource; verified supplier/project profile; community-service contribution; factual media expert availability. Every offer requires capacity, rights and claims approval.

## Pipeline
Research organization → verify relevance and actual relationship → record value offered and disclosure/cost → approve contact and message → only a later authorized phase contacts → deliver value → request the natural action (resource citation, partner profile, case-study attribution, supplier listing, referral introduction) → record result and disclosure. Reject if requested action controls anchor text, hides payment, creates a false endorsement or exposes client information.

## Qualification
Audience fit, legitimate organization, editorial discretion, local/industry relevance, real traffic/referrals, current maintenance, transparent cost, proof needs, workload and reputational risk. Owner approves every outreach target and spend.
