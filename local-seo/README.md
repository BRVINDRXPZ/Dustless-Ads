This directory contains the internal, approval-gated operating system for Google Business Profile, local pages/content, reviews, project media, citations/NAP, local links/authority, schema, map observations, Search Console, QA, and the first 90 days of execution.

Start with `LOCAL_SEO_STRATEGY.md`, resolve `NAP_STANDARD.md` and `GBP_AUDIT_TEMPLATE.md`, then follow `90_DAY_LOCAL_SEO_ROADMAP.md`. CSVs are working registers; Markdown files define governance and workflows.

Nothing here is approved public copy or authorization to publish, submit, request, contact, upload, connect, spend, or modify a live system. Current platform facts must be validated from dated official sources and owner-controlled interfaces. Customer addresses, credentials and secrets do not belong in this repository.
