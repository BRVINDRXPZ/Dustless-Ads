# Local Content Cluster

`LOCAL_CONTENT_CLUSTER.csv` is the production backlog. It treats topics as hypotheses, not verified search volume.

## Production standard
Answer the question directly; distinguish general education from site-specific advice; cite current primary technical/manufacturer/government sources where applicable; use New Orleans context only when technically reviewed; disclose variables; avoid guaranteed outcomes, absolute dust/health claims and price fabrication. Each article must earn its own intent, link to one principal service page and related guides, and receive owner/expert/claims review.

## Cluster paths
1. **Save/repair:** saveability, replace vs refinish, stains/water, matching, leaks/storms → repair/restoration.
2. **Process/finish:** sanding expectations, finish types/sheen, screen-and-coat, prep, furniture, occupancy, cure → refinishing.
3. **Local materials/buildings:** humidity, pine/heart pine, historic homes → proof-gated historic page.
4. **Partner planning:** commercial phasing, contractor coordination → evidence-gated partner pages.
5. **Buyer confidence:** mistakes, pricing factors, contractor questions → estimate CTA without vendor disparagement.

## Proof gate
No named property, neighborhood, material/species diagnosis, product result, schedule, customer quotation or before/after appears without job record, expert verification and written permission. Update technical guidance on a documented review cycle.
