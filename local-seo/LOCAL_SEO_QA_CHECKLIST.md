# Local SEO QA Checklist

## Truth, access and policy
- [ ] Owner-approved NAP version/effective date; every unknown remains blocked.
- [ ] Public name matches real-world brand; no keyword/city stuffing.
- [ ] Address/service-area eligibility and privacy documented; no fake/virtual office.
- [ ] Owner/manager access, 2FA/recovery/least privilege and duplicate/suspension evidence reviewed without secrets.
- [ ] Current official category, service, post, Q&A/contact, media and review rules saved with date.

## GBP change package (not execution)
- [ ] Before screenshots/export and rollback plan.
- [ ] Categories reflect actual work; area list reflects operations.
- [ ] Phone/website/appointment links owned, monitored and tested.
- [ ] Regular/special hours owner assigned.
- [ ] Description/services/attributes factual, deduplicated, within current limits.
- [ ] Media rights, privacy, color and crop pass; before/after is same project.
- [ ] UTM syntax/final URL/call/forms/consent/CRM attribution pass.
- [ ] Written owner approval identifies exact fields; capacity confirmed.

## Reviews
- [ ] Universal eligible cohort; no sentiment screen, incentive, pressure or supplied review voice.
- [ ] Direct link valid; one-reminder maximum proposal; opt-out works.
- [ ] Sensitive reviews escalated; responses factual, private, calm and approved.
- [ ] No review/aggregate schema unless visible, supported and policy-eligible.

## Pages/schema/technical
- [ ] Unique intent, content, proof, links and CTA; no doorway/clone/cannibalization.
- [ ] Titles/H1/canonical/status/indexability/sitemap/breadcrumb/mobile/performance/accessibility pass.
- [ ] NAP and area-served data match approved truth; no customer address.
- [ ] Schema validates and matches visible page; current official eligibility checked.
- [ ] Phase 5 events fire once, contain no PII and reconcile to CRM after approved implementation.

## Citations, links and authority
- [ ] Platform/organization legitimate; cost/terms/membership/address-display verified.
- [ ] Existing duplicates/conflicts recorded before net-new submission.
- [ ] No paid-link, reciprocal quota, spam network, false affiliation or manipulated anchor.
- [ ] Outreach/project/media/customer permissions and claims evidence approved.

## Measurement and release
- [ ] Manual baseline uses fixed matrix and personalization caveat; no invented positions.
- [ ] Search Console access/report plan approved; manual/security issues escalated.
- [ ] QA evidence, executor, monitoring, rollback and owner signature recorded.
- [ ] Separate explicit authorization exists before any public action. **Phase 6 authorization: none.**
