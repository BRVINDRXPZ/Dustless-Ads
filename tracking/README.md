# Phase 5 Measurement Architecture

This directory specifies events, attribution, calls, forms, CRM stages, offline outcomes, consent, spam/deduplication, QA, and implementation order. It is vendor-neutral design work only: no account, ID, credential, customer record, live tag, endpoint, or CRM connection is present.

Start with `MEASUREMENT_PLAN.md`, then implement only in the gated order in `IMPLEMENTATION_SEQUENCE.md`. `CONVERSION_DICTIONARY.csv` is the event source of truth. Phase 4 remains paused until every QA and owner-approval gate passes.
