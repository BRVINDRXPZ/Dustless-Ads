# NOLA Dustless Repository Instructions

This repository is the business, marketing, sales, content, analytics, and growth operating system for NOLA Dustless Hardwood Floors.

Before beginning any task:

1. Read AGENTS.md.
2. Read MASTER_DIRECTIVE.md.
3. Read BUSINESS_CONTEXT.md.
4. Read any files directly relevant to the current task.
5. Check OWNER_INPUT_REQUIRED.md for unresolved questions.

## Working rules

- Produce usable deliverables, not generic advice.
- Work incrementally.
- Preserve previous approved decisions.
- Clearly distinguish verified facts, assumptions, proposals, and placeholders.
- Update CHANGELOG.md after substantial work.
- Update NEXT_ACTIONS.md when priorities change.
- Add unresolved questions to OWNER_INPUT_REQUIRED.md.
- Keep files organized in the appropriate repository folder.
- Prefer concise, actionable documentation over bloated filler.
- Never invent reviews, testimonials, awards, licenses, certifications, project history, prices, performance figures, customer quotes, or company claims.
- Never publish advertisements.
- Never spend money.
- Never contact leads, customers, contractors, vendors, or partners.
- Never change a live website, advertising account, CRM, analytics account, or social account without explicit owner approval.
- Never expose passwords, tokens, customer data, API keys, or private business information.
- Create changes on a new branch and prepare a pull request.
- Include a clear summary of all changes and any owner decisions required.

The company should become:

- The first flooring company homeowners think about.
- The hardwood specialist contractors call automatically.
