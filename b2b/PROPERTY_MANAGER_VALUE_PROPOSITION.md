# Property Manager Value Proposition

> **Internal draft — owner approval required.** This is an operating specification, not authorization to contact, publish, connect a CRM, promise capacity, or make a public claim. Bracketed fields and `TBD—owner verification required` are not public copy. Use only public business contact data or permissioned records.

## Position
An accountable evaluation path for suitable occupied, turnover and common-area hardwood needs: triage scope, access/occupancy, resident communication ownership, repeatable documentation, schedule constraints, maintenance options and closeout.
## Controls
Portfolio does not equal automatic priority. Qualify unit/common-area type, recurrence, notice/access, furniture, pets, parking, protection, payment/PO, vendor rules, emergency expectations and capacity. No emergency response, unit-volume or schedule promise is currently approved.
