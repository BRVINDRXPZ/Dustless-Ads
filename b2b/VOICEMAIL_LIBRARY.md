# Voicemail Library

> **Internal draft — owner approval required.** This is an operating specification, not authorization to contact, publish, connect a CRM, promise capacity, or make a public claim. Bracketed fields and `TBD—owner verification required` are not public copy. Use only public business contact data or permissioned records.

## Warm / requested
“Hi [Name], [Name] with NOLA Dustless at [approved number]. [Introducer/context] suggested I call about [specific hardwood need]. No need to call if it is not relevant; otherwise I’d welcome a brief fit conversation. Again, [number].”
## Cold business context
“Hi [Name], [Name] with NOLA Dustless at [number]. I’m calling because [specific public business relevance], not with a generic list. I have one question about how you evaluate hardwood partners. If useful, call [number]; otherwise no response is needed.”
## Bid / project
“Hi [Name], calling about [project ID], specifically [clarification]. We have not assumed or reserved [scope/schedule]. Please call [number] or reply through the established project channel.”
One voicemail per sequence unless a live project requires agreed coordination.
